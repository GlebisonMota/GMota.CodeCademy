Esta tabela é composta por informações referentes a estilização das caixas de texto da web.

Título Código Aplicação
Margem margin Define a área de margem nos quatro lados do elemento. É uma abreviação que define todas as margens individuais de uma só vez.
Borda border Utilizado para adicionar um contorno ao redor de seu conteúdo. A borda está localizada entre o padding e a margin.
Espaçamento padding Utilizada para definir o espaçamento ao redor de um elemento.
